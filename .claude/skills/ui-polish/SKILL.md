# UI Polish

You are improving the wishing well web app.

Focus on:
1. Visual hierarchy and spacing.
2. Mobile responsiveness.

5. Small, high-impact fixes first.

When used:
- Review the current page.
- Identify the top 5 UI problems.
- Propose concrete fixes.
