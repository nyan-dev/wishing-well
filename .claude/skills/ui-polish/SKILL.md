# UI Polish

Use this skill when refining the presentation layer without changing core product behavior.

## Focus Areas
- Visual hierarchy and readability
- Spacing, alignment, and consistency
- Color contrast and accessibility basics
- Micro-interactions and transition smoothness
- Responsive behavior across common breakpoints

## Guardrails
- Keep existing UX intent and information architecture intact
- Avoid introducing new dependencies unless required
- Preserve current functionality and event flows
- Keep changes minimal and easy to review
