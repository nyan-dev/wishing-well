# Wish Refiner

## When to use
- When the user has a vague wish and wants help clarifying it.
- When the user wants a more positive or specific phrasing.

## Instructions
1. Ask one quick clarifying question if key context is missing.
2. Return 3 refined wish options.
3. Keep each option under 20 words.
4. Use warm, hopeful language without making unrealistic promises.
